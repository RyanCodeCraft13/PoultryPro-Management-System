# 🐔 Poultry Farm Management System

A complete Poultry Farm Management System built using **PHP & MySQL**.  
This system helps poultry farm owners manage batches, egg production, feeds, sales, expenses, and profit monitoring.

---

## 🚀 Features

✅ Secure Admin Login  
✅ Dashboard Overview  
✅ Chicken Batch Management (Broiler / Layer)  
✅ Egg Production Recording  
✅ Feed Inventory Tracking  
✅ Sales Recording  
✅ Expense Tracking  
✅ Automatic Profit Calculation  
✅ Simple & Clean Interface  
✅ XAMPP Ready  

---

## 🛠️ System Requirements

- XAMPP / WAMP / LAMP
- PHP 7+
- MySQL
- Web Browser (Chrome recommended)
